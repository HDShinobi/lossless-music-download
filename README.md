# Lossless Music

Android app to download verified-lossless music (FLAC) via decentralized JS extensions,
with a DLNA/UPnP + WebDAV library server. Built on the SpotiFLAC engine architecture.

- Package: `xyz.losslessmusic.app`
- Status: in development (Phase 0 bootstrap complete)
- Design & plans: `docs/superpowers/specs/`, `docs/superpowers/plans/`

> For educational/research purposes. `go_backend/` is vendored from SpotiFLAC (MIT) and kept pristine; see `docs/`.
